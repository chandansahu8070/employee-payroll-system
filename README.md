# Employee Payroll System (Java - OOPS)

## 📖 Project Description
This is a simple **Employee Payroll System** implemented in **Java**.  
It demonstrates core **Object-Oriented Programming (OOPS) concepts** such as:
- **Encapsulation** – Employee details stored in private fields with getters & setters.  
- **Inheritance** – `FullTimeEmployee` and `PartTimeEmployee` extend the base `Employee` class.  
- **Polymorphism** – `calculateSalary()` method is overridden based on employee type.  
- **Abstraction** – Interface `Payable` defines a contract for payroll calculation.  

---

## ⚡ Features
- Add employee details (Name, ID, Type).  
- Calculate salary for both full-time and part-time employees.  
- Demonstrates OOPS principles in action.  
- Console-based mini project (easy to run & understand).
